## Drone Delivery

# Como Rodar o Projeto
git clone --branch master https://github.com/gustavocravo/DroneDeliveryAPI.git

# Back-end
Abra a pasta backend, dentro da pasta clique no arquivo .sln para iniciar o projeto no Visual Studio 2022 (ou na sua IDE de preferência) e pressione F5 para rodar o projeto. A API será iniciada automaticamente.

#Front-end
Abra a pasta frontend no seu terminal (Git Bash, Visual Studio Code Terminal, etc.).
Instale as dependências:
npm install
Inicie a aplicação:
npm run dev
Acesse a URL fornecida no seu navegador.
